# PermissionTools
Library for simplify android permissions management

## Gradle Dependency (jitpack.io)

project level _build.gradle_
```Gradle
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

app _build.gradle_
```Gradle
dependencies {
    implementation 'com.github.rSquared-software:live-utils:x.x.x'
}
```
